🔧 GUI 資料來源修正計劃

版本 1.0 · 2026-06-05 · 根因：GUI 從錯誤來源讀取公開資源

一、架構確認（Repo B 的邊界）

Repo B（{user}/tech-rebalance，private）只包含用戶私有資料： 
{user}/tech-rebalance/
├── accounts.json          ← 用戶帳戶設定（唯一真正私有）
├── data/{account_id}/     ← 每日交易狀態（自動產生）
└── .github/workflows/daily.yml  ← 技術必要物，內容由引擎標準化
pub engine（tech-rebalance-pub，public）擁有所有公開定義： 
tech-rebalance-pub/
├── brokers/alpaca.json     ← 券商規格
├── brokers/tradier.json
├── strategies/*.json       ← 策略定義
├── schemas/*.json          ← 資料 schema
└── manifest.json           ← 版本索引（含 brokers/strategies 清單）

四、修改計劃

M-01：重寫 catalog.py — 公開資源全改讀 pub engine

修改前（錯誤）

def list_brokers(self):
    result = _stems(self.store.list_dir("brokers"), "broker-schema")  # ← 讀 Repo B
    if result: return result
    ...fallback...

def list_strategies(self):
    result = _stems(self.store.list_dir("strategies"), "schema")      # ← 讀 Repo B
    if result: return result
    ...fallback...

def broker_spec(self, broker_id):
    return self.store.read_json(f"brokers/{broker_id}.json")          # ← 讀 Repo B

修改後（正確）

def list_brokers(self):
    m = self._effective_manifest()                   # pub engine manifest
    return mf.brokers(m) if m else _FALLBACK_BROKERS

def list_strategies(self):
    m = self._effective_manifest()                   # pub engine manifest
    return mf.strategies(m) if m else _FALLBACK_STRATEGIES

def broker_spec(self, broker_id):
    m = self._effective_manifest()                   # pub engine manifest
    return (m.get("brokers") or {}).get(broker_id, {})

def broker_environments(self, broker_id):
    m = self._effective_manifest()
    if m: return mf.broker_environments(m, broker_id)
    return _FALLBACK_ENVS.get(broker_id, ["paper", "live"])
去掉所有 store.list_dir("brokers") / store.list_dir("strategies") / store.read_json("brokers/*.json")

M-03：schedule_view.py — 動態掃 workflow

# 修改前（硬寫）
_WORKFLOWS = [".github/workflows/daily_all_accounts.yml",
              ".github/workflows/momentum_tool_v2.yml"]

# 修改後（動態）
def _list_workflows(self):
    try:
        names = self.store.list_dir(".github/workflows")
        return [f".github/workflows/{n}" for n in names if n.endswith(".yml")]
    except Exception:
        return []
